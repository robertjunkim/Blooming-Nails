# Blooming Nails Bothell Website

## Deploy to Vercel

1. Go to https://vercel.com
2. Create a new project
3. Drag and drop this folder OR upload the ZIP
4. Click Deploy

Your website will automatically go live.

## Replace Booking Link
Search for:
https://squareup.com/us/en/appointments

Replace it with your actual Square booking URL.

## Add Your Own Photos
Replace the Unsplash image URL in index.html with your own salon photos.
